This is an apache struts 2 application, which presents a simple company model.
We make use of the struts2.2.1 library.
See http://struts.apache.org/download.cgi#struts221
Make sure, that your struts library is located in /struts/WebContent/WEB-INF/lib/
Copy or drag your project to your servlet container (e.g. Tomcat V7.0, see http://tomcat.apache.org/download-70.cgi) and startup the servlet container.
In a web browser go to: http://localhost:8080/struts/
You should see a web page with a list of departments.
